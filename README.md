# Library-Management-System
Library Management System GUI in C++ using gtkmm 

As a part of CSE 1325 Homework project. 

## Build and Run
```
make main
./lms
```

## Unit Tests
```
make test_pub
make test_lib
```

## Features
* Add Publications
* Add Patrons
* List Publications
* List Patrons
* Pre-Fill Data
* Check-In
* Check-Out
